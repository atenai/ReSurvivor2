# ReSurvivor2

「Reサバイバー2&モーションビュアー」

（概要・テーマ）
この作品は、Re:サバイバーの続編として開発中の作品です。
伝令である主人公が情報を目的地まで戦場をくぐりぬけて行く「伝令」というテーマで制作しています。
またゲームシステムは前作の横スクロールシューティングゲームからTPSゲームへの進化を目標にしています。
その開発過程でアニメーションのチェック用にモーションビュアーを作成しました。

(開発時期・期間)
2023年11月～2025年2月(25ヶ月)

（開発ツール）
・ゲームエンジン：Unity(バージョン2022.3.12f1)
・言語：C# 
・開発環境：Visual Studio Code 
・3D データ： Unityのアセットストアを使用しています。
・2D データ : フリー素材を使用しています。

（チームメンバー）
開発チーム（1人）
・プログラマー1人（担当）

(Reサバイバー2操作方法)
[キーボード]
W・A・S・D：移動
マウス：カメラ
左クリック：射撃
右クリック：エイム
R：リロード
1：ハンドガン
2：アサルトライフル
3：ショットガン
4・G：地雷
5・E：防弾プレート（体力回復薬）
6・Q：食料（スタミナ回復薬）
Shift：ダッシュ
M：マップ
F：インタラクト
P・ESC：ポーズ
T：マウスカーソル

(モーションビュアー操作方法)
[キーボード]
右クリック + マウス：カメラ
左クリック：各種ボタンクリック
ESC：ゲーム終了

(Reサバイバー2起動方法)
「ReSurvivor2(ビルド)」というフォルダーを開いてください。
その中にある、「ReSurvivor2.exe」というアプリケーションをダブルクリックしてください。

(モーションビュアー起動方法)
「MotionViewer(ビルド)」というフォルダーを開いてください。
その中にある、「ReSurvivor2.exe」というアプリケーションをダブルクリックしてください。
